SEO Page Generator generates Pages depending of your used Keywords and Server capacities.

== Description ==

All this Plugin needs is a List of Keywords, a Title and a Content (optional Meta Title and Description for YOAST SEO metas), to generate Pages and links themself with each other.

The Title and the Content have to contain a Placeholder which has to be defined by you - this Placeholder will be replaces in COMPLETE Title and Content, but isn't case sensitive.


== Installation ==

1. Download the Plugin and Install it.
2. Deactivate automatic adding of new Pages to Menus
3. Fill in the Page Contents
4. Generate your new SEO Optimized Pages


== Frequently Asked Questions ==

= Will Pages generated with this Plugin appear in the Menu? =
Yes, it will. But you can uncheck the Option to add all new Pages to a Menu - then this pages will only be visible through the Sitemap.
